# Redesign Metrics Experience — Phase 1

**[▶ View Live Preview](https://ouracheal.github.io/redesign-metrics-experience-phase1/)**

A prototype for the redesigned metrics experience, including:

- **Home** — unified NL search that routes to the right view
- **Metrics Explorer** — Datadog-style inline query builder with NL query parsing and Bits Assistant panel
- **Metric Summary** — metric detail and health overview
- **Governance** — cost, cardinality, and usage workflows
